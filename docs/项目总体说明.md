# 基于卷积神经网络的MNIST数据分类系统 - 项目总体说明

## 一、项目概述

本项目实现了一个完整的基于卷积神经网络的MNIST手写数字识别系统，包含基础要求和5个创新扩展功能。

**项目特点：**
- ✅ 完整实现所有基础要求（A-E）
- ✅ 5个创新扩展（非标准卷积、轻量化、对抗鲁棒性、可解释性、增量学习）
- ✅ 工程化项目结构
- ✅ 详细文档和说明
- ✅ Web端演示程序

## 二、功能模块

### 2.1 基础要求

#### A. 训练网络并记录损失值
- 实现完整的训练循环
- 记录每个batch的损失值
- 绘制损失曲线

#### B. 改变卷积层和池化层数
- 测试1层、2层、3层卷积
- 对比不同层数的准确率
- 分析层数对性能的影响

#### C. 改变卷积核大小
- 测试3×3、5×5、7×7卷积核
- 对比不同核大小的准确率
- 分析感受野的影响

#### D. 分析有效性
- 生成详细分析报告
- 对比实验结果
- 总结规律和结论

#### E. 数据分类系统
- 完整的训练和测试流程
- 模型保存和加载
- 性能评估

### 2.2 创新扩展

#### 创新1：非标准卷积与注意力机制
**实现内容：**
- SENet-CNN（Squeeze-and-Excitation注意力）
- DilatedCNN（空洞卷积）
- DepthwiseCNN（深度可分离卷积）

**对比维度：**
- 准确率
- 参数量
- 训练时间
- 推理时间

**文档：** `docs/非标准卷积与注意力机制说明.md`

#### 创新2：轻量化与移动端部署
**实现内容：**
- UltraLightCNN（<50k参数，>98%准确率）
- ONNX模型导出
- Flask Web端演示程序

**特点：**
- 参数量仅32,934
- 模型大小132KB
- 实时手写识别

**文档：** `docs/轻量化与移动端部署说明.md`

#### 创新3：对抗鲁棒性测试
**实现内容：**
- FGSM攻击（单步）
- PGD攻击（多步迭代）
- 对抗训练防御

**测试内容：**
- 不同网络结构的鲁棒性
- 不同扰动强度的影响
- 防御方法效果

**文档：** `docs/对抗鲁棒性测试说明.md`

#### 创新4：可解释性分析
**实现内容：**
- Grad-CAM热力图生成
- 多层注意力分析
- 不同架构对比

**分析维度：**
- 不同卷积层的关注区域
- 层数对注意力的影响
- 卷积核大小对注意力的影响

**文档：** `docs/可解释性分析说明.md`

#### 创新5：增量学习与数据增强
**实现内容：**
- EWC增量学习（防止遗忘）
- 弹性变形数据增强
- 随机擦除数据增强
- MixUp混合增强

**实验场景：**
- 先训练0-4数字，再学习5-9
- 对比有/无EWC的遗忘率
- 对比不同数据增强策略

**文档：** `docs/增量学习与数据增强说明.md`

## 八、使用说明

### 8.1 训练自定义模型

```python
from src.models import ConfigurableCNN
from src.main import get_data_loaders
from src.train import run_training

# 加载数据
train_loader, test_loader = get_data_loaders()

# 创建模型
model = ConfigurableCNN(num_conv_layers=2, kernel_size=3)

# 训练
losses, accuracies = run_training(
    model, train_loader, test_loader,
    epochs=10, learning_rate=0.001
)
```

### 8.2 使用预训练模型

```python
import torch
from src.lightweight_models import UltraLightCNN

# 加载模型
model = UltraLightCNN()
model.load_state_dict(torch.load('experiment_results/lightweight_cnn.pth'))
model.eval()

# 推理
with torch.no_grad():
    output = model(input_tensor)
```
