# 增量学习与数据增强说明文档

## 一、实验目的

1. 实现EWC增量学习方法，在学习新类别时防止遗忘旧知识
2. 设计创新的数据增强策略（弹性变形、随机擦除、MixUp），提升模型泛化能力

## 二、理论基础

### 2.1 增量学习问题

**灾难性遗忘（Catastrophic Forgetting）：**
神经网络在学习新任务时，会大幅降低在旧任务上的性能。

**场景模拟：**
- 旧任务：识别数字0-4
- 新任务：识别数字5-9
- 目标：学习新任务的同时保持旧任务性能

### 2.2 EWC方法（Elastic Weight Consolidation）

**核心思想：** 通过Fisher信息矩阵识别重要参数，在训练新任务时约束这些参数不大幅变化。

**损失函数：**
```
L_total = L_new + (λ/2) Σ F_i(θ_i - θ*_i)²
```

**参数说明：**
- L_new：新任务损失
- F_i：参数i的Fisher信息（重要性）
- θ*_i：旧任务训练后的参数值
- λ：正则化系数

**Fisher信息矩阵计算：**
```
F_i = E[(∂log p(y|x,θ) / ∂θ_i)²]
```

### 2.3 数据增强策略

#### 2.3.1 弹性变形（Elastic Deformation）

**原理：** 通过随机位移场对图像进行非线性变形，模拟手写时的自然变化。

**实现步骤：**
1. 生成随机位移场 dx, dy
2. 高斯滤波平滑位移场
3. 应用位移场变形图像

**参数：**
- alpha：变形强度（推荐36）
- sigma：平滑度（推荐4）

#### 2.3.2 随机擦除（Random Erasing）

**原理：** 随机选择矩形区域并擦除，提高模型对遮挡的鲁棒性。

**参数：**
- probability：应用概率（0.5）
- sl, sh：擦除区域面积范围
- r1：宽高比范围

#### 2.3.3 MixUp

**原理：** 将两个样本及其标签按比例混合。

**公式：**
```
x_mix = λ·x_i + (1-λ)·x_j
y_mix = λ·y_i + (1-λ)·y_j
```

**参数：**
- alpha：Beta分布参数（推荐1.0）

## 三、实验设计

### 3.1 增量学习实验

**对比实验：**
1. 无EWC：直接在新任务上训练
2. 使用EWC：添加Fisher约束训练

**评估指标：**
- 旧任务准确率（训练前/后）
- 新任务准确率
- 遗忘率 = 旧任务准确率下降幅度

### 3.2 数据增强实验

**对比策略：**
1. 无增强（基线）
2. 弹性变形
3. 随机擦除
4. MixUp
5. 组合增强（弹性+擦除+MixUp）

**评估指标：**
- 测试准确率
- 泛化能力提升

## 五、实验结果分析

### 5.1 EWC增量学习效果

**典型结果：**

| 指标 | 无EWC | 使用EWC | 改善 |
|-----|-------|---------|------|
| 旧任务准确率（训练前） | 95.2% | 95.2% | - |
| 旧任务准确率（训练后） | 45.8% | 88.7% | +42.9% |
| 新任务准确率 | 94.3% | 92.1% | -2.2% |
| 遗忘率 | 49.4% | 6.5% | -42.9% |

**结论：**
- EWC显著降低遗忘率（约85%改善）
- 新任务准确率略有下降（trade-off）
- 整体性能大幅提升

### 5.2 数据增强效果

**准确率对比（5个epoch后）：**

| 策略 | 准确率 | 提升 |
|-----|--------|------|
| 无增强 | 98.2% | - |
| 弹性变形 | 98.6% | +0.4% |
| 随机擦除 | 98.4% | +0.2% |
| MixUp | 98.7% | +0.5% |
| 组合增强 | 99.0% | +0.8% |

**结论：**
- 所有增强策略均有效提升准确率
- MixUp效果最显著
- 组合增强效果最佳

### 5.3 增强样本可视化

**弹性变形效果：**
- 数字形状自然变化
- 保持可识别性
- 模拟真实手写变化

**随机擦除效果：**
- 部分区域被遮挡
- 提高遮挡鲁棒性
- 防止过拟合

## 八、应用价值

### 8.1 增量学习应用

1. **持续学习系统：** 不断学习新知识而不遗忘
2. **个性化模型：** 适应用户特定需求
3. **资源受限场景：** 无需重新训练全部数据

### 8.2 数据增强应用

1. **小样本学习：** 扩充训练数据
2. **提升泛化：** 增加数据多样性
3. **防止过拟合：** 正则化效果

## 十、常见问题

**Q1: EWC的lambda如何选择？**
A: 通常在1000-10000之间，需要根据任务调整。过大会限制新任务学习，过小会导致遗忘。

**Q2: 为什么弹性变形适合手写数字？**
A: 模拟了真实手写时的自然变化，增加了数据的真实性和多样性。

**Q3: MixUp会不会产生不合理的样本？**
A: MixUp生成的是"软标签"样本，虽然视觉上不自然，但能有效提升泛化能力。

**Q4: 增量学习能否应用到其他任务？**
A: 可以，EWC是通用方法，适用于分类、检测等多种任务。
