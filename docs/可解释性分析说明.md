# 可解释性分析说明文档

## 一、实验目的

使用Grad-CAM技术生成模型注意力热力图，分析不同网络结构（层数、卷积核大小）对图像关键区域的关注度，提升模型的可解释性。

## 二、理论基础

### 2.1 Grad-CAM原理

**Grad-CAM (Gradient-weighted Class Activation Mapping)** 是一种可视化技术，通过梯度加权生成类激活映射。

**核心步骤：**
1. 获取目标卷积层的特征图 A^k
2. 计算目标类别对特征图的梯度
3. 全局平均池化得到权重 α^k
4. 加权求和：L = ReLU(Σ α^k · A^k)

**公式：**
```
α^k_c = (1/Z) Σ_i Σ_j (∂y^c / ∂A^k_ij)
L^c_Grad-CAM = ReLU(Σ_k α^k_c · A^k)
```

### 2.2 为什么需要可解释性

1. **调试模型：** 发现模型关注错误区域
2. **建立信任：** 理解模型决策依据
3. **改进设计：** 指导网络结构优化
4. **符合法规：** 满足AI可解释性要求

## 三、实验设计

### 3.1 多层注意力分析

分析同一模型不同卷积层的注意力模式：
- 浅层：关注边缘、纹理等低级特征
- 中层：关注局部形状
- 深层：关注整体语义

### 3.2 架构对比分析

对比不同网络结构的注意力：
- 1层 vs 2层 vs 3层
- 3×3 vs 5×5 vs 7×7卷积核

### 3.3 统计指标分析

计算注意力热力图的统计指标：
- 平均注意力强度
- 中心区域注意力
- 边缘区域注意力
- 高注意力区域占比

## 五、实验结果分析

### 5.1 层级特征分析

**观察规律：**
- **第1层：** 关注笔画边缘，注意力分散
- **第2层：** 关注数字主体，注意力集中
- **第3层：** 关注整体形状，注意力更聚焦

### 5.2 网络结构影响

**层数影响：**
- 1层网络：注意力分散，容易受噪声干扰
- 2层网络：注意力适中，平衡性好
- 3层网络：注意力高度集中，但可能过拟合

**卷积核大小影响：**
- 3×3：关注局部细节
- 5×5：关注中等范围特征
- 7×7：关注全局结构

### 5.3 典型案例

**数字"8"的注意力模式：**
- 模型主要关注两个圆圈的连接处
- 边缘笔画获得较高注意力
- 内部空白区域注意力低
