## 三、实验内容详解

### 3.1 实验A：训练网络并记录损失值

#### 实验目的
训练卷积神经网络，记录每个batch的损失值，观察训练过程中损失的变化趋势。

#### 实现方式
```python
# 在 src/train.py 中实现
def train_epoch(model, train_loader, criterion, optimizer, device):
    model.train()
    all_losses = []
    
    for batch_idx, (data, target) in enumerate(train_loader):
        data, target = data.to(device), target.to(device)
        
        optimizer.zero_grad()
        output = model(data)
        loss = criterion(output, target)
        loss.backward()
        optimizer.step()
        
        # 记录每个batch的损失值
        all_losses.append(loss.item())
    
    return all_losses
```

#### 输出结果
- **损失曲线图**: `experiment_results/loss_curve_B.png` 和 `loss_curve_C.png`
- **损失数据**: 保存在JSON文件中，包含每次迭代的损失值

#### 分析要点
1. 损失值应随训练逐渐下降
2. 曲线应趋于平稳，表示模型收敛
3. 不同网络配置的损失下降速度可能不同

### 3.2 实验B：改变卷积层和池化层数

#### 实验目的
对比不同网络深度对分类准确率的影响，理解网络层数与模型性能的关系。

#### 实验配置

| 配置 | 卷积层数 | 池化层数 | 卷积核大小 | 参数量 |
|------|---------|---------|-----------|--------|
| 配置1 | 1层 | 1层 | 3×3 | ~804k |
| 配置2 | 2层 | 2层 | 3×3 | ~421k |
| 配置3 | 3层 | 3层 | 3×3 | ~241k |

#### 网络结构示例

**1层卷积网络**:
```
输入(1×28×28) → Conv1(32) → ReLU → MaxPool → Flatten → FC(128) → FC(10)
```

**2层卷积网络**:
```
输入(1×28×28) → Conv1(32) → ReLU → MaxPool 
              → Conv2(64) → ReLU → MaxPool 
              → Flatten → FC(128) → FC(10)
```

**3层卷积网络**:
```
输入(1×28×28) → Conv1(32) → ReLU → MaxPool 
              → Conv2(64) → ReLU → MaxPool 
              → Conv3(128) → ReLU → MaxPool 
              → Flatten → FC(128) → FC(10)
```

#### 运行方式
```bash
# 运行完整实验
python experiments.py

# 或单独运行实验B
python -c "from experiments import ExperimentRunner; runner = ExperimentRunner(epochs=10); runner.experiment_b_layer_comparison()"
```

#### 预期结果

| 层数 | 预期准确率 | 训练时间（CPU） | 特点 |
|------|-----------|----------------|------|
| 1层 | 87-89% | ~10分钟 | 参数多，表达能力有限 |
| 2层 | 91-93% | ~15分钟 | 平衡性能与效率 |
| 3层 | 92-94% | ~20分钟 | 表达能力强，参数少 |

#### 分析要点
1. **层数增加的优势**:
   - 更强的特征提取能力
   - 更少的参数量（通过逐层降维）
   - 更高的分类准确率

2. **层数增加的挑战**:
   - 训练时间增加
   - 可能出现梯度消失问题
   - 需要更多的训练数据

3. **最佳实践**:
   - 对于MNIST这样的简单任务，2-3层卷积足够
   - 更深的网络需要配合BatchNorm、残差连接等技术

### 3.3 实验C：改变卷积核大小

#### 实验目的
对比不同卷积核大小对分类准确率的影响，理解感受野与特征提取的关系。

#### 实验配置

| 配置 | 卷积核大小 | 网络层数 | 感受野 | 参数量 |
|------|-----------|---------|--------|--------|
| 配置1 | 3×3 | 2层 | 小 | ~421k |
| 配置2 | 5×5 | 2层 | 中 | ~455k |
| 配置3 | 7×7 | 2层 | 大 | ~504k |

#### 感受野分析

**3×3卷积核**:
- 单层感受野: 3×3
- 两层感受野: 5×5
- 特点: 捕捉局部细节特征

**5×5卷积核**:
- 单层感受野: 5×5
- 两层感受野: 9×9
- 特点: 平衡局部与全局特征

**7×7卷积核**:
- 单层感受野: 7×7
- 两层感受野: 13×13
- 特点: 捕捉更大范围的特征

#### 运行方式
```bash
# 运行完整实验
python experiments.py

# 或单独运行实验C
python -c "from experiments import ExperimentRunner; runner = ExperimentRunner(epochs=10); runner.experiment_c_kernel_comparison()"
```

#### 预期结果

| 卷积核 | 预期准确率 | 参数量 | 特点 |
|--------|-----------|--------|------|
| 3×3 | 91-93% | 421k | 参数少，适合细节 |
| 5×5 | 91-93% | 455k | 平衡性能 |
| 7×7 | 90-92% | 504k | 参数多，可能过拟合 |

#### 分析要点
1. **小卷积核（3×3）**:
   - 参数量少，计算效率高
   - 通过堆叠可以获得大感受野
   - 现代CNN的主流选择

2. **大卷积核（7×7）**:
   - 单层即可获得大感受野
   - 参数量多，容易过拟合
   - 计算量大

3. **最佳实践**:
   - 对于28×28的MNIST图像，3×3或5×5卷积核最合适
   - 可以用多个小卷积核替代大卷积核

### 5.2 关键指标解读

**1. 准确率（Accuracy）**
- 定义: 正确分类的样本数 / 总样本数
- 范围: 0-100%
- 目标: >90%（基础要求），>95%（优秀）

**2. 损失值（Loss）**
- 定义: 模型预测与真实标签的差异
- 趋势: 应随训练逐渐下降
- 收敛: 最终应稳定在较低值

**3. 参数量（Parameters）**
- 定义: 模型中可训练参数的总数
- 影响: 影响模型大小、训练速度、内存占用
- 权衡: 在准确率和效率之间找平衡

### 5.3 常见问题分析

**问题1: 准确率低于预期**
- 可能原因: 训练轮数不足、学习率不合适、模型容量不够
- 解决方案: 增加训练轮数、调整学习率、增加网络层数

**问题2: 损失不下降**
- 可能原因: 学习率过大或过小、梯度消失、数据问题
- 解决方案: 调整学习率、检查网络结构、验证数据加载

**问题3: 过拟合**
- 表现: 训练准确率高，测试准确率低
- 解决方案: 添加Dropout、数据增强、减少模型复杂度

## 七、常见问题FAQ

### Q1: 如何加速训练？
**A**: 
1. 使用GPU加速（需要CUDA支持的PyTorch）
2. 增大batch_size（如果内存允许）
3. 减少训练轮数（可能影响准确率）
4. 使用更小的模型

### Q2: 数据集下载失败怎么办？
**A**:
1. 检查网络连接
2. 手动下载MNIST数据集放到 `data/MNIST/raw/` 目录
3. 使用国内镜像源

### Q3: 如何保存训练好的模型？
**A**:
```python
import torch

# 保存模型
torch.save(model.state_dict(), 'my_model.pth')

# 加载模型
model = ConfigurableCNN(num_conv_layers=2, kernel_size=3)
model.load_state_dict(torch.load('my_model.pth'))
model.eval()
```

### Q4: 如何在自己的图片上测试？
**A**:
```python
from PIL import Image
import torch
from torchvision import transforms

# 加载图片
img = Image.open('my_digit.png').convert('L')

# 预处理
transform = transforms.Compose([
    transforms.Resize((28, 28)),
    transforms.ToTensor(),
    transforms.Normalize((0.1307,), (0.3081,))
])
img_tensor = transform(img).unsqueeze(0)

# 预测
with torch.no_grad():
    output = model(img_tensor)
    pred = output.argmax(dim=1)
    print(f"预测结果: {pred.item()}")
```

### Q5: 实验结果与预期不符怎么办？
**A**:
1. 检查随机种子设置
2. 确认数据加载正确
3. 验证模型结构
4. 多次运行取平均值
5. 查看训练日志排查问题

本基础实验通过系统的对比实验，验证了卷积神经网络在MNIST手写数字分类任务上的有效性。实验结果表明：

1. **网络深度**: 适当增加网络层数可以提高准确率，但需要权衡训练成本

3. **系统设计**: 模块化的系统设计便于扩展和维护
